Project Utopia Windows launcher bundle

1. Double-click "Project Utopia.cmd".
2. The launcher starts a local server and opens the game in Edge or Chrome app mode.
3. To enable live AI, place a .env file next to the launcher, based on .env.example.

Notes:
- The launcher prefers Microsoft Edge and falls back to Google Chrome.
- The local server stops automatically after the browser app window closes.
