## Beta Release Notes

### Quick Review

First pillar: colony-scale NPC AI with runtime LLM support and deterministic fallback. Since HW04, this pillar was upgraded into a hierarchical pipeline (`Perceiver -> Planner -> Executor -> Evaluator`) with memory, skill reuse, and evaluation feedback.

### Secondary Pillar Integration

Second pillar: pathfinding and crowd navigation are now integrated into the main loop rather than running as separate demos.

- A* now uses elevation cost plus road/bridge bonuses, so terrain and infrastructure directly change worker routes.
- The planner can place roads between production buildings and warehouses, so AI decisions now reshape logistics.
- Boids tuning, job reservation, and occupancy-aware scoring reduce clumping and make worker execution more reliable.

### New Features

- Added 6 terrain generators and deeper terrain rules for elevation, moisture, fertility, weather, drought, and wildfire.
- Added terrain-aware colony planning with specialized placement and evaluation feedback.
- Added logistics upgrades including road-network reasoning and smarter worker targeting.
- Added broader benchmark coverage for behavior probes and crisis scenarios.

### Known Issues

- Live LLM mode still requires a valid local API key; fallback mode remains stable without it.
- Wildlife balance is conservative and tends to settle at a low population floor.
- The production build still has a large `vendor-three` chunk to optimize before final submission.
