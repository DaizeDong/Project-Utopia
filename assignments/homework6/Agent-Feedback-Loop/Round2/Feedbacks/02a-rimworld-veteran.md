# Project Utopia 评测 · RimWorld 老兵视角（Round 2）

## 玩家身份自述

2000 小时 RimWorld 老手，顺带 DF 精神病患者一枚。我进这款游戏抱着两个期待：第一，能在高难下看到 AI 殖民者像 Prepare Carefully 出来的一样有性格；第二，资源链深到让我掏出 Excel 建产能模型。Banished 的硬核食物平衡、ONI 的压力管道、DF 的 Z-level 都是我日常的饭点。今天点开 Project Utopia，我是当作一款 "Web 上能跑的 mini-RimWorld" 来的，所以我会苛刻。

---

## Session 1：Temperate Plains —— 被动观察者的八分钟

载入页面，背景已经是一张活着的温带平原，河流横穿地图，一眼看下去湖是蓝格子 ASCII 贴图，草地是纯色绿块——可以接受，我容忍美术。下方写着 "Broken Frontier · frontier repair · 96×72 tiles"，副标题叫我 "重建西侧伐木线、夺回东侧仓库"。OK，像个 RimWorld 事件剧本。我按 Start Colony。

进入游戏后我愣了一下：**地图正中央已经摆着一大坨建筑**——4 座农场、7 座围墙、2 座伐木营、7 个仓库，工人 13 名，食物 110，木材 12，石材 5。没有 "从零开始放地基" 的过程，开局直接给我一个残缺的基地。这不是 RimWorld 的 naked brutality，更像是一个 "托管场景 + 玩家偶尔微调" 的半自动沙盒。右上角冒出一个 DRIFT 标签，写着 "autopilot: colony holding steady"——autopilot！？我还没动手它就自己在跑了。

我开了 4× 快进，点了 Colony 面板。1 分 34 秒，Food 从 110 掉到 29，净产率 **0.0/min**——这是食物产线完全断裂的信号。Wood 已经 0，Stone 仅剩 1，Meals/Medicine 全是 0。人口反而从 13 涨到 19（visitors 4 转正或某种事件），职业分布里 COOK=0、HERBALIST=0、HAUL=1——典型的前期职业结构失衡。我立刻想建一个 Kitchen，点开成本发现要 **8 wood + 3 stone**，而我两样都约等于零，拉不动。

2 分 55 秒，第一个亡魂出现："Nell-23 died (starvation)"。Food 归 0。我改点 Farm（5 wood）—— 但 wood 也没了，地基放不下去。RimWorld 在这种时刻能做狩猎、采野莓、屠宰、吃皮革，而这里**除了干瞪眼我啥也做不了**。游戏 HUD 突然从 "7 warehouses built" 变成 "0 warehouses built"，同时 Farms 从 4 跌到 1 —— 我怀疑建筑在资源归零时被系统判定拆解了，或者这里有某种 "维护费" 机制，但完全没有 UI 告诉我发生了什么。DF 塌房时你至少能从 announcements 里看到 Urist McStone 的死因和拆解物，这里只有一行 "[158.2s] Nell-23 died (starvation)" 冷冰冰丢出来。

4 分 23 秒，Brinn-13 也走了，Workers 13 人。5 分 30 秒 Vela-8、6 分左右 Ilia-12——**死亡走廊式加速**。到 8 分 05 秒殖民地勉强稳住在 Food 16 / Workers 8 / 4 farms / Dev 39，分数 350，AI 的 DIRECTOR 在念 "route repair and depot relief" —— 这其实就是开场那段场景目标，系统始终围着一个没完成的 "depot reclaimed 0/1" 打转。那个 depot 我点了半天也没找到视觉标记。

**Session 1 结论**：我感觉自己像在看一段 AI 跑 benchmark 的回放，而不是在玩一个游戏。我的 "输入 → 反馈" 闭环几乎失效。

---

## Session 2：Rugged Highlands —— 换汤不换药

我回到标题页切换模板到 **Rugged Highlands**，点 Start。结果……**一模一样的地图**。同样的蓝河，同样的中央预置基地，同样的 "Broken Frontier — Reconnect the west lumber line" 副标题，同样的 4 农 / 7 墙 / 2 伐木。**没有山脊，没有高程差，没有石场密集带**——Highlands 该有的所有视觉和玩法特征一个都没有。HUD 右上角和剧本名也没换。

我一度以为自己没点对，刷新重来再次选 Rugged Highlands——还是同一张图。对照 RimWorld 的 Large Hills / Mountains biome，那玩意不仅改变地形、还改变动植物、温度、矿层和敌人类型。这里连**地表色都没变**。

我继续观察到 1 分 58 秒：Food 0 / Wood 0 / Stone 0 / Workers 20 / Farms 13 —— 13 座农场，食物还是零，说明 farm 要么需要更复杂的前置（种子？土壤？旱季？）要么这些 farm 根本没铺在 "可耕地" 上。UI 没有任何一个地方告诉我农场的 yield、土壤品质、灌溉状态——不像 Banished 会告诉你 crop yield %，不像 ONI 会告诉你 plant 是否 cultivated/wild。

**Session 2 结论**：三个模板里至少两个是贴牌。这对声称有 "6 maps" 的游戏是致命的信任破产。

---

## Session 3：Archipelago Isles —— 第三次同一张地图

我抱着最后一丝希望选了 **Archipelago Isles**——群岛诶，应该有海、有桥、有跨海物流。结果你猜怎么着：**还是那张破 Broken Frontier 温带平原图**。标题预览里的水域形状和前两次像素级一致，描述依然是 "Reconnect the west lumber line"。这不是 RNG 碰巧，这是**模板选择器压根没生效**（或者至少在我测的 build 上没生效）。

我按 F1 开了 Help，面板整齐地列出了 Controls / Resource Chain / Threat & Prosperity / What makes Utopia different 四页——写得很认真，"Open Build panel, place a Farm, add a Lumber Mill..." —— 新手引导的文案是游戏里最成熟的部分。可讽刺的是，这些建议**在我当前这局根本没法执行**，因为 wood 早就归零，建不起 farm。

我按 L 想看 Heat Lens（supply-chain 热力图）——视觉上**没有任何变化**。画面没出现色块、没出现连线、什么都没有。按 L 两次三次都是同样的死寂。RimWorld 的 Fertility Overlay / Beauty Overlay / Temperature Overlay 一打开立刻把地图涂成彩虹，这里的 "Heat Lens" 对我来说是个完全不可见的功能。

**Session 3 结论**：核心卖点之一（供应链可视化）在这个 build 上没有可见反馈。

## 系统深度评价（老兵视角）

**AI / Director 系统**：右上角会在 DRIFT 和 DIRECTOR 两个标签间切换，后者会念一段像 LLM 生成的宏观目标描述（"route repair and depot relief: the colony should sustain route repair and depot relief while keeping hunger and carried cargo from overriding the map's intended reroute press"）——这种措辞说明背后确实有一个 prompt/planner，但**它说的话跟玩家没关系**，既不告诉我哪里缺物流，也不标出被它 prioritize 的 tile。DF 的 Noble demands、RW 的 Tynan event log 都有具体 actionable info，这里只有抽象 LLM flavor text。

**工人 AI / 职业分配**：Colony 面板里 WOOD=10、FARM=5、SMITH=1、COOK=0、HERBALIST=0、HAUL=1 这种分布意味着**职业不是 RimWorld 那种优先级矩阵，而是一个被动统计口径**。我找不到 "把 X 号工人指派为 Cook" 的入口，没法手动 pin job priority，没法设 schedule，没法设 role restrictions。这在 colony sim 里是降维打击级的功能缺失。

**Entity Focus**：侧栏下方提示 "Click a worker, visitor, or animal on the map to inspect it here"。我尝试点工人位置，但因为当前工具是 Farm/Road，左键变成 "放建筑"，worker 选中根本触发不了——操作模态互锁得死板。切换到 Erase 再点工人？Erase 更直接拆建筑。没有一个 "select / deselect tool" 的中性模式。老兵的本能：按 Q / Esc 退出工具栏。按 Esc 的确清选择，但再点工人依然无反应。最终我整局没成功打开过任何一个工人的 detail 面板，无从看 Mood、Skills、Relationships、Trait——这些是 RimWorld 的灵魂，Utopia 这里是黑盒。

**时间控制**：Pause / Play / 4× 三档。没有 1/2 速、没有 3×、没有"自动暂停条件"（RimWorld 的 pause-on-event 是救命功能）。4× 在我实测下真实流速约为**游戏内 41 秒 / 真实 15 秒**，比例大约 2.7×，离标称 4× 有差距。对长期运营，这个倍速远远不够——RimWorld 的 Superspeed (3x) 在 Dev Mode 下甚至能拉到 6×。

**建筑经济**：资源只有 food/wood/stone/herbs 四种基础 + meals/tools/medicine 三种加工——数量上对标 Banished 的 log/stone/food，深度远低于 RimWorld 的 steel/component/plasteel/uranium/chemfuel 体系。Kitchen 8 wood + 3 stone 的成本对比开局只给 12 wood / 5 stone 的预算，**只够建 1 座加工设施**，而我还要同时维持 farm / lumber / warehouse 的持续替换——死亡螺旋在设计上几乎写死了。

**场景 / 任务**：HUD 永远挂着 "1/1 supply routes open, 0/1 depots reclaimed"。我三局都没搞懂 depot 具体在哪、怎么算 "reclaim"、reclaim 完会不会换下一个目标。目标设计缺乏 feedback。

## 与 RimWorld / DF 的直接对比

| 维度 | RimWorld | DF | Project Utopia |
|-----|----------|----|----|
| 殖民者个性 | Traits / Backstory / 好感 / Mental break | 更丰富，带情绪短语 | **完全缺失**，只有编号名 |
| 工作分配 | Priority matrix | Labor noble + room 设置 | 无手动分配 |
| 生物多样性 | 80+ 动物 | 100+ | Herbivores 2 / Predators 1（数字 = 种类？个体？含糊） |
| 地形差异 | 10+ biome, 深度不同 | Z-level + 地质 | **3 个模板实测视觉完全相同** |
| 事件系统 | Tynan 事件链 | Caravan / siege | "Broken Frontier" 单一 scripted event |
| 中后期目标 | 飞船 / 殖民章程 / DLC | Fortress legends | "Survive as long as you can" 无尽 |
| 信息 UI | Inspector / Storytime / Logs | Announcement history | 一行 log `[t] NPC died (cause)` |
| 速度档位 | 4 档 + pause-on-event | 极细 | 3 档 |
| 地图大小 | 75×75 到 300×300 | Embark 任意 | 固定 96×72 |

Utopia 在**只有 RimWorld 骨架的 10%-15%** 的水准上挣扎。

## 最想让开发者修复的 5 件事

1. **修 template 选择器**：Rugged Highlands / Archipelago Isles / Coastal Ocean 必须真实改变地形、资源分布、敌对势力，否则请把下拉菜单去掉——虚假宣传的伤害大于没有选择。
2. **开场资源 / 成本再平衡**：开局 12 wood 不足以支持 Kitchen（8+3）或 Warehouse 的即时建设，第一次资源断链几乎不可逆。要么加开场资源，要么让 farm/lumber 回产周期更短，要么加一个 "饥荒时工人自发采野果" 的 safety net。
3. **工人微管理入口**：必须让玩家点工人弹出 detail 面板（mood / skill / carry / current action），并能设职业优先级。否则这是放置游戏而不是 colony sim。
4. **Heat Lens 要有可见反馈**：按 L 后至少叠加一层色温 / 连线 / 标记；现在按了等于没按，功能形同虚设。
5. **Director 文案改 actionable**：LLM 吐 "autopilot: colony holding steady" 不如告诉我 "West lumber line broken: road gap at (42, 36), place Road here"。老兵要的是坐标，不是 flavor text。

## 评分理由

**3 / 10**。

- 加分（+3）：新手 Help 面板写得很用心，资源链概念、暂停 / 快进 / 建造 UI 的基础骨架都在。HUD 数据面 (Dev index、Score、Survived time、resource/min) 证明底层 simulation 确实在跑，不是假数据。
- 扣分（−7）：三个模板实测是同一张图；工人 detail / 职业分配完全无法交互；开场经济几乎强制崩溃且无救灾手段；Heat Lens 无可见效果；Director 文案与可操作无关；4× 速度实测未达标；场景目标（depot/route）缺乏定位提示。

作为 RimWorld 老兵，我玩了 20 分钟，没有一次获得"我救活了这局"的心流时刻。殖民地自己生自己死，我只是在旁边点按钮。**这不是 colony sim，这是 colony benchmark replay viewer**。愿意再打开看下一个版本，但绝不会装在 Steam 库里给朋友安利。

一句话定性：**骨架有，血肉缺；模板假，操作僵，救灾无门，DF 老兵一律劝退。**
