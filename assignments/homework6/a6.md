## Peer Playtesting

**Important Note**

I substituted 10 independent LLM agents for 5 human classmates. **Here are my reasons:**

1. I was inspired by a recent research project ([karpathy/autoresearch](https://github.com/karpathy/autoresearch)) that uses a pure agent loop to automatically tune parameters during neural network training. I was curious whether a similar approach could work for game development.
2. Game development turns out to be a perfect fit for such an agentic loop: player feedback drives game improvements, which in turn generate better feedback. Peer playtesting maps cleanly onto this cycle.
3. No social-desirability bias. The available classmate pool has little colony-sim domain knowledge, while LLM agents can be prompted into specialist personas (RimWorld veteran, speedrunner, indie critic, etc.) on demand.

**Here is my design:**

The pipeline runs four stages per round, all implemented with Claude Agent SDK + Playwright headless browser:

- **Reviewer** (Stage A): Blind agents with browser access only. No source code, docs, changelog, or prior round scores. Each reviewer plays independently and produces a structured Markdown feedback file.
- **Enhancer** (Stage B): One agent per feedback file, reading only that feedback plus the repo code. Produces an implementation plan with root-cause analysis and file-level step specifications. No cross-reviewer communication.
- **Implementer** (Stage C): Serial execution, one plan per commit. No changes outside the accepted plan scope.
- **Validator** (Stage D): Runs the full test suite, long-horizon benchmark, and browser smoke check. Benchmark regressions cannot be offset by UI improvements.

Key constraints: reviewers stay blind across rounds (no delta summaries, no prior scores); all agents are isolated from each other (anti-echo-chamber); each feedback file produced a 10–20 KB structured review with reproduction steps and DOM audits.

**The results turned out to be quite effective:**

1. Efficient: 9 rounds completed within 3 days.
2. Agent prompts and raw reviews are fully reproducible and checked in.
3. Major improvement: average score rose from 3.1 / 10 (Round 0) to 5.5 / 10 (Round 9).
